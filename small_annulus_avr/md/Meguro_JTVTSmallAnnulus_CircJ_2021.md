> [!NOTE] Key Message
> **日本全国 J-TVT registry 5,870 例**で 11.0% (647例) が小弁輪 (annulus area ≤314 mm²)。**小弁輪群の 93.2% が女性、BSA 1.34 m²**。小弁輪は PPM の独立予測因子 (OR 1.84, P<0.001)。同 cohort 内 SAPIEN 3 20mm (BE, intra-annular) vs Evolut R 23mm (SE, supra-annular) 比較で **SE が iEOA・mPG 両指標で有意に良好** (iEOA 1.07 vs 0.94, P=0.001; mPG 11 vs 14 mmHg, P<0.001)。**日本人小弁輪 TAVR で supra-annular SE が hemodynamics 上有利**を全国規模で裏付け。

## 1. 背景・問題設定

- アジア人は欧米に比して低身長・低体重で aortic annulus が小さい。TAVR が普及した日本で、小弁輪の頻度・短期アウトカム・PPM の規模を全国レジストリで明らかにする必要があった。
- 既報 (Yashima OCEAN-TAVI) は extremely small (≤314 mm²) を扱い、本研究は同基準を全国 5,870 例で適用。
- 同サイズ outer casing 同士 (BE 20mm vs SE 23mm) を直接比較し、現実的な device 選択指針を提示。

## 2. 患者集団

- N=5,870 (J-TVT 全 10,761 例から MDCT・echo データ完備の cohort)
- 期間 2013年8月-2017年12月, 103 施設
- 小弁輪 (≤314 mm² MDCT) 647 例 (11.0%) vs 通常 5,223 例
- 小弁輪群: **女性 93.2%, 年齢中央値 85 歳, BSA 1.34 m²**, 身長 145 cm, 体重 45 kg
- STS-PROM 6.7% (小弁輪) vs 6.2% (通常), P<0.001
- LVEF 66% (小弁輪) > 63% (通常)
- 弁種: SAPIEN XT 40.3%, SAPIEN 3 22.4%, CoreValve 9.7%, Evolut R 27.0%
- 弁サイズ (小弁輪群): **20mm 25.7%, 23mm 48.4%, 26mm 21.6%, 29mm 3.9%** (29mm は anatomy 上不適合だが小数記載あり)

## 3. 手技詳細

- 標準的 transfemoral 主体 (81.9%)
- 標準 TAVR 手技、heart team で device 選択
- サブ解析: **SAPIEN 3 20mm (n=90) vs Evolut R 23mm (n=103)** = 同程度 outer casing の比較

## 5. 議論ポイント

- **小弁輪日本人女性 (BSA 1.34 m²) は身長 145 cm, 体重 45 kg と極小体格**
- SAA で PPM 18% は SAVR の 45-93% より遥かに低い → TAVR は sewing ring がないため EOA 確保有利
- Cover index (oversizing) が SAA で高い → PVL が逆に少ない（trade-off）
- SE supra-annular (Evolut R 23mm) は同等 outer casing の BE (SAPIEN 3 20mm) より hemodynamics 圧倒的良好
- SAPIEN 3 20mm の severe PPM 7.8% vs Evolut R 23mm 2.9% (NS, n 不足) — Lee Korean Circ J 2021 と方向性一致
- **PVL evaluation が施設依存**で過大評価の可能性あり

## 6. 本研究RQへの含意（最重要）

- **Primary RQ**: 日本人 TAVR 全国規模で小弁輪 11.0% (≈ 90%が女性, BSA 1.34 m²)。**SAVR で小弁輪に AAE せず 19mm 弁を使うベンチマーク**として、TAVR の方が PPM 圧倒的少 (18% vs SAVR 45-93%) → 高齢日本人小弁輪では TAVR 第一選択になりつつあるエビデンス。
- **Secondary RQ (19mm 許容性, initial sizing)**: SAPIEN 3 20mm の severe PPM 7.8%, Evolut R 23mm 2.9% → 同等 anatomy なら **SE 23mm supra-annular が intra-annular BE 20mm より優位**。AAE の代替戦略として「device 選択での up-sizing」が成立。
- **Tertiary RQ (Asian-specific 戦略)**: ★ **J-TVT の日本人小弁輪 cohort は世界最小級の BSA 1.34 m²** で、Western cohort の Mauri/Rogers/TAVI-SMALL とは body size が大きく異なる。Korean Lee cohort BSA 1.5 m² より更に小さい。device 選択の hemodynamics 効果が体格別にどう変動するかは未解明だが、本論文は **「日本人 SAA で SE supra-annular の優位性は維持される」**ことを示した重要データ。

## 7. 限界

- Echocardiography が core lab でなく施設実施 → PVL 過大評価可能性
- 30 日アウトカムのみで long-term durability/SVD は不明
- Selection bias: SAPIEN 3 20mm vs Evolut R 23mm は同等 anatomy ではあるが、operator preference が confounder
- 116 例の SAPIEN 3 20mm が EOA 未登録で除外 (post-marketing 義務期間の差) → 系統的偏倚は否定的だが完全除外不可
- AAE-SAVR との直接比較は不可（TAVR registry のみ）
