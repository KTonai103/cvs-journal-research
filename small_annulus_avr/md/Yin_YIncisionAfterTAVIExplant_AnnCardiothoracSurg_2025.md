> [!NOTE] Key Message
> **TAVR (size 23 SAPIEN) explant 後の Y-AAE 適応 case vignette**: 74 歳女性 BAV、TAVR 2 年後に severe AI + moderate stenosis (mean PG 28、peak PG 47)、annulus 19mm。**Y-AAE で 19mm → size 25 bioprosthesis 留置**、postop mean PG 7 mmHg。Univ Michigan の 119 例 cohort で operative mortality 0.8%、PPM 0 を背景に、TAVR-explant + Y-AAE が ViV-TAVR 失敗例の salvage 戦略として確立。

---

## 1. 背景・問題設定
VIVA trial で small annulus は TAVR の hemodynamic 優位性消失、moderate/severe PPM が >30% に達する device もある。BAV + 19mm annulus 患者で TAVR initial intervention は再考の余地。本症例で TAVR 失敗後の salvage として Y-AAE を選択。

## 2. 患者集団
- N=1 case vignette: 74 歳女性、BAV stenosis 既往
- TAVR 2 年前 (size 23 SAPIEN 3, Edwards)
- 現症状: severe AI + moderate stenosis (mean PG 28, peak PG 47)
- Native annulus 19mm
- Cohort 引用 (Hassler 2024 Semin Thorac CardioVas Surg PedCardAnnu): n=119、BAV 45% (54/119)、operative mortality 0.8%、stroke 0.8%、PPM 0/119、postop AVA 2.3 cm²、mean PG 6-7 mmHg

## 3. 手技詳細
- Proximal aortic arch + RA two-stage cannulation
- Antegrade + retrograde cardioplegia、ice slush
- Aorta transection 2cm above STJ
- SAPIEN deformed but incorporated in root、sharp dissection で除去
- Severe PVL at basal ring を確認、native valve tissue 切除、annulus debridement、LVOT mild damage、annulus 19mm 測定
- **Standard Y-incision** (LCC-NCC commissure → aorto-mitral curtain → 両 trigone まで)
- 2x3 inch Hemashield patch を 4cm 幅にトリム、**Arc modification** 採用、4-0 Prolene running
- Annulus 19mm → **size 25 bovine pericardial valve** 留置可能化
- Non-pledgeted 2-0 Ethibond、strut を L-R commissure に
- **Roof technique** で aortotomy closure (2cm 後方 longitudinal aortotomy)

## 5. 議論ポイント
- TAVR の hemodynamic 優位性は small annulus で消失 (VIVA)、PPM rate >30% in some devices
- 19mm + BAV 患者の initial intervention は SAVR + Y-AAE が theoretically 優位
- TAVR-explant 後 salvage options: (1) ViV-TAVR (本症例で不可と判断) (2) isolated SAVR (3) SAVR + AAE
- Matched annulus (≤23mm) 研究: AAE + SAVR は isolated SAVR と比べ **mod/severe PPM 19% vs 31%、6y survival 98% vs 74%** (Makkinejad 2025 ATS)
- Y-incision は Nicks/Manouguian に比べ 3 size up 可能・hemodynamics 優位
- **Y-AAE を small annulus 患者の initial 第一選択**として推奨

## 6. 本研究RQへの含意（最重要）
- **Primary RQ**: 19mm 狭小弁輪 BAV 患者で Y-AAE size 25 留置・mean PG 7 mmHg を実現。**TAVR failure 後 salvage としても Y-AAE が有効**を示し、small annulus lifetime management の最適戦略を再考させる重要 case。
- **Secondary RQ (19mm 許容性、ViV用 initial sizing)**: 19mm annulus に対し size 23 SAPIEN を選択した initial TAVR の妥当性を疑問視。**"Y-AAE で size 25 留置すれば最初から実現できた hemodynamics"** を明示。
- **Tertiary RQ (Asian-specific)**: 米国 case だが、19mm + BAV は Asian 女性で頻発する patterns。本 case の戦略 (TAVR の代わりに SAVR+Y-AAE first) は Asian で更に relevant。

## 7. 限界
- Single case vignette
- Long-term outcome なし
- TAVR explant の technical complexity は症例ごと
- Cohort 数値 (119 例 mortality 0.8%) は Hassler 2024 からの引用、本 case report は anecdotal
