> [!NOTE] Key Message
> VIVID registry 292 例の aortic ViV で **TAVR device implantation depth と post-procedural gradient の関係** を中央 fluoroscopy lab 評価。**High implantation で elevated gradient (≥20 mmHg) が低減**: CoreValve Evolut 15% (high) vs 34.2% (low), p=0.03; Sapien XT 18.5% vs 43.5%, p=0.03。**最適 depth: CoreValve Evolut 0–5mm, Sapien XT 0–2mm (0–10% frame height)**（sensitivity 91.3%/88.5%）。Device position が最強の独立予測因子 (OR 0.22)、device type と外科弁失敗 mechanism も影響。**ViV-TAVR では high implantation が hemodynamic 成功の鍵—初回 SAVR 小弁植込みの hemodynamic 弱点を mitigate する手技的補正**。

---

## 1. 背景・問題設定
- ViV-TAVR の主限界は THV underexpansion による高 gradient と severe PPM。
- Supra-annular 位置の THV leaflet が外科弁 ring の非弾性的制約から逃れられるなら高 implant が好ましいはず、しかし malposition / migration リスクも。
- 本論は **最適 implant depth を device 別に定義** することを目的。

## 2. 患者集団
- N = 292（CoreValve Evolut 157, Sapien XT 135）。
- Mean age 78.9±8.7 yr、男性 60.3%。
- 外科弁 small (label ≤21mm): CoreValve 46.5%, Sapien XT 23%。
- 失敗 mechanism: stenosis 42–45%, mixed 29–34%, regurgitation 24–26%。

## 3. 手技詳細
- Central fluoroscopic corelab (St. Paul's Hospital) で implant depth を計測。
- Depth: surgical valve ring の ventricular border からの距離（mm; CoreValve）または %frame height（Sapien XT）。
- ROC 解析で最適 cutoff 決定。

## 5. 議論ポイント
- Supra-annular SE device (CoreValve Evolut) は intraannular Sapien XT より gradient 低い（device 効果 OR 0.5）。
- Stenosis/mixed mode failure は regurgitation より高 gradient リスク（OR 3.12）。
- High implantation は migration や PPI risk と trade-off だが本研究では mortality / coronary obstruction / pacemaker 差なし。

## 6. 本研究RQへの含意（**最重要**）
- **Primary RQ — CatF 中心**: ViV-TAVR の hemodynamic 成功は **device 選択 (CoreValve/Evolut) + 高 implant** が両輪。
- **Secondary RQ — initial SAVR sizing と将来 ViV**:
  - 本論は CoreValve label ≤21mm が 46.5%—**small surgical valve を持つ症例で high implant 戦略は有効**だが、それでも 15% は elevated gradient → **初回 SAVR で 21mm 以下は確実に hemodynamic 不利**。
  - 「small valve への ViV」は high implantation で mitigation 可能だが、limit に近い。**AAE で初回 ≥23mm 弁植込みが ViV 余地確保で望ましい**。
  - Stentless or BVF (bioprosthetic valve fracturing) との組合せで更なる gradient 低減が後続研究で確立。
- **CatE 比較視点**: 初回 TAVI を small annulus で行うと、再 ViV-in-TAVI で「TAVI ステント ring」内に implant 必要—本論の「high implant within stented surgical valve」と類似の物理制約。
- **Tertiary RQ (Asian-specific)**: Asian で small valve への ViV 多発が予想 → 本論の implant depth recommendation は Asian operator に直接 actionable knowledge。

## 7. 限界
- 観察研究、fluoroscopy 評価で 3D 情報不足。
- 旧世代 device (CoreValve, Sapien XT)、Evolut Pro+ / Sapien 3 では結果異なる可能性。
- Long-term durability data なし。
- Stentless / non-fluoroscopic marker 弁は除外、generalizability 限定。
