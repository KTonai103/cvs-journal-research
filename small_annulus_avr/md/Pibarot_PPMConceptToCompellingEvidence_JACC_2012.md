> [!NOTE] Key Message
> Pibarot/Dumesnil による Rahimtoola 1978 概念から 2012 までの **30 年 PPM 総説 (editorial)**。**Severe iEOA ≤0.65 / Moderate 0.65-0.85 の cutoff** が 27/34 (79%) の研究で採用され ASE/EACVI 2009 guideline で endorse。Head meta-analysis 2012 (27,186 pts) と Takagi 2012 で **severe PPM は long-term mortality に重大影響、moderate PPM も影響あり** という consensus を強調。**Projected EOAi (manufacturer chart 不可、reliable reference 使用) で術前予測すべき**。**AAE は古典的対応**だが、modern prosthesis 設計で 30 → 1% に severe PPM 削減可能 (Bleiziffer)。

## 1. 背景・問題設定

- Rahimtoola 1978 (Circulation) で PPM 概念提唱から 30 年。
- Daneshvar/Rahimtoola 2012 (JACC viewpoint) に対する **Pibarot/Dumesnil の編集委員コメンタリー**。
- 中心議題: (1) PPM 測定法, (2) 重症度評価, (3) 評価時期, (4) outcome impact, (5) 予防戦略。

## 2. PPM 測定 — Indexed EOA (iEOA)

- **Indexed EOA (EOA / BSA)** が唯一の妥当パラメータ
- EOA は実際の血流が通る面積で、geometric orifice area や sewing ring diameter と異なる
- 同じ size 23 でも bioprosthesis 種類で reference EOA は **1.6-2.0 cm²** と大きく異なる
- **Projected EOAi (実装前計算)** vs Measured EOAi (術後実測)
  - Bleiziffer 2007 → Projected EOAi が PPM 予測のベスト
  - Florath 2008 (stentless 限定で否定的) には限界あり
- Manufacturer chart は overoptimistic, **clinical trials の reliable data set 推奨**

## 3. PPM Severity Cutoff (consensus)

| Category | iEOA (cm²/m²) | 採用率 |
|---|---|---|
| **Severe PPM** | **≤0.65** | 79% of 34 studies (Head meta) |
| **Moderate PPM** | **0.65-0.85** | 同 |
| No PPM | >0.85 | - |

- 2009 ASE/EACVI guideline 採用
- 肥満補正は別途 (severe ≤0.55, moderate 0.55-0.70 in obese, Mao 2026 引用)

## 4. PPM の長期 Outcome 影響

| Study | n | Findings |
|---|---|---|
| **Head et al. EHJ 2012** | **27,186 (34 studies, 133,141 patient-years)** | **severe PPM HR 1.84, moderate 1.19; overall 1.34** |
| Takagi 2012 (Int J Cardiol) | meta-analysis | severe PPM late mortality 影響 |
| Blais 2003 (Circulation) | - | 30-day mortality 増加 |
| Mohty 2009 (JACC) | - | Age, obesity, LVD で differential heterogeneity |
| Flameng 2010 (Circulation) | - | PPM → 早期 SVD |
| Ruel 2006 / Tasca 2006 / Walther 2006 / Urso 2009 | - | LV dysfunction 群で PPM 影響強い |

**Vulnerable subset** (PPM 影響大):
- 低 LVEF
- 重度 concentric hypertrophy
- 中等度以上 mitral regurgitation 併存
- 極重症 baseline AS

## 5. 評価時期

- ASE/EACVI 2009 recommendation: **退院後 2-4 週**の TTE が baseline
- 6-12 ヶ月再評価で reference value 比較 (pathological healing, intrinsic dysfunction 除外)

## 6. 予防戦略

### 1978 (Rahimtoola 提唱時)
- **AAE が唯一の選択肢**

### 2012 (本論文時点)
- **改良 prosthesis design (supra-annular, hemodynamic 良好弁) が主体**
- Bleiziffer 2007: 戦略変更で moderate PPM 44% → 30%, severe PPM 9% → 1%
- 予防の trade-off: AAE による operative risk 増・弁 durability 影響を考慮
- **Projected EOAi 計算で「小弁輪 + 小体格なら小型弁 OK」を validation** (Yoshikawa 2008 Japanese 引用)

## 7. アウトカム (本研究の論評範囲)

| Outcome | Result |
|---|---|
| 30-day mortality | severe PPM (projected EOAi) は独立予測因子 |
| Severe PPM long-term all-cause mortality | Head meta HR 1.84 |
| Moderate PPM | HR 1.19 (有意, ただし vulnerable subset で増強) |
| SVD acceleration | Flameng 2010 で確認 |

## 8. 議論ポイント

- 「PPM clinical relevance 争議」は Rahimtoola 2012 viewpoint で Daneshvar が再提起 → 本 editorial で反論
- Measured EOAi のみでは survival bias (perioperative death 患者は echo なし)
- Projected EOAi で **個別患者で術前から PPM 予測可能**
- 「アジア人小体格では projected EOAi が low risk → 小型弁 OK」の方向性も提示 (Yoshikawa 引用)
- AAE 一律不要、**個別判断 (PPM severity × baseline risk)**

## 9. 本研究RQへの含意（最重要）

- **Primary RQ**: AAE-SAVR vs no AAE vs TAVI 比較において、**「modern prosthesis 改善で AAE 必要性は減少 (Bleiziffer 9 → 1%)」**は本研究 RQ の根本論拠。本 editorial は「AAE を一律にせよ」とも「するな」とも言わず、**「projected EOAi で個別判断」**を提唱。
- **Secondary RQ (19mm 許容性, initial sizing)**: ★★ **「projected EOAi 計算で small valve を validate 可能」**は、本研究 RQ の secondary を直接サポート。19mm valve の reference EOA (例: SJM Regent 19mm ≈ 1.5-1.8 cm²) を BSA 1.4 m² で割れば iEOA ≈ 1.07-1.29 → severe PPM 回避可能。Initial sizing の理論基盤。
- **Tertiary RQ (Asian-specific 戦略)**: ★ Yoshikawa 2008 (Japanese, small St. Jude long-term) を引用し、**「日本人小体格 + 小弁輪は projected EOAi で計算すれば AAE 回避 OK」**との示唆。Daneshvar/Pibarot editorial が **アジア人小体格戦略のエビデンス基盤を提供**。
