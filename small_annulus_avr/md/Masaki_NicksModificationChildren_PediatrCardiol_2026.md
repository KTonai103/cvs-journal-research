> [!NOTE] Key Message
> **宮城こども病院 + 岩手医大 Masaki らの日本人小児症例**: 9 歳男児 (BSA 1.0 m²、annulus 21mm)、severe AI に対し **modified Nicks 法 (small Y-shape 切開 LCC 側 5mm、NCC 側 7mm + 12mm 幅 J-Graft patch)** で 21mm valve から 21mm St Jude Regent 留置可能化、3 年無症状。Yang Y-incision の "3-5 sizes up" は小児で過剰侵襲との立場から **1 size up minimum 侵襲** 設計。

---

## 1. 背景・問題設定
小児 AVR では somatic growth と long-term observation を考慮し surrounding tissues への interference を最小化したい。Nicks/Manouguian/Konno/Yang の中で Manouguian (mitral leaflet) と Konno (ventricular septum) は長期影響懸念。Yang Y-incision は 3-5 size up 可能だが、LCC 広範切開で left coronary kinking risk、NCC 広範切開で mitral valve 長期 deterioration 懸念、subaortic stenosis に対応せず。本症例で **Nicks の改良 (small Y-shape)** で安定 1 size up を実現。

## 2. 患者集団
- N=1: 9 歳男児
- 身長 132 cm、体重 29 kg、**BSA 1.0 m² (小児)**
- Severe AI、LV end-diastolic 59mm 拡大
- Aortic annulus 21mm (TTE)
- 三尖弁 AV (cusps thickened、RCC tethered to center → AI)

## 3. 手技詳細
- Median sternotomy、bicaval drainage + ascending aortic return
- Antegrade + selective antegrade cardioplegia
- Leaflets resection、St Jude Medical Regent valve sizing → 19mm tightly through、**21mm 不通過**
- Aortotomy を NCC 左側から annulus 下へ延長
- **Small Y-shape incision in subaortic curtain**: LCC 側 5mm、NCC 側 7mm
- 12mm 幅 rectangular patch (J-Graft、Japan Lifeline Inc) を 7-0 Prolene running で縫合
- 21mm sizer 通過、23mm tight → **21mm St Jude Regent 留置**（intra-annular position）
- Patch 上 4 sutures は outside-inside、aortotomy を patch trim で closure

## 5. 議論ポイント
- **日本人男性での 19mm 機械弁は PPM 発症リスクあり** → 小児男児では将来体格不明だが 21mm 以上推奨と明記
- Nicks 単独では「achieved enlargement insufficient」事例あり、改良が必要
- Manouguian は MV、Konno は septum を侵襲 → 長期影響懸念
- Yang Y-incision の批判点:
  - LCC 広範切開で **left coronary kinking** risk
  - NCC 広範切開 + mitral annulus 含む patch 縫合で **長期 MV function deterioration**
  - 3-5 size up を可能にするが **subaortic stenosis 未対応**
- 理論: 1 size up は annulus diameter +2mm、circumference +6mm → **10mm 幅 patch** で sufficient (margin 含めて 12mm)
- 本手技は Yang の "minimum invasion" 派生版として位置づけ

## 6. 本研究RQへの含意（最重要）
- **Primary RQ**: **BSA 1.0 m² 小児 + annulus 21mm** に対する modified Nicks (small Y) は狭小弁輪 SAVR の代替戦略。AAE-without-aggression という critical view を提供。
- **Secondary RQ (19mm 許容性)**: 「**日本人男性で 19mm 機械弁は PPM リスク → 21mm 以上を目指す**」と明示。これは本研究 RQ の secondary RQ（19mm 許容性）に直接答えを与える日本人著者の立場。Future ViV は明示せず（小児では機械弁長期使用前提）。
- **Tertiary RQ (Asian-specific)**: **完全に日本人小児症例**。Asian/Japanese で 21mm 達成のための小規模 AAE 戦略を提示。Yang の "aggressive 3-5 size up" 派と Masaki の "minimal 1 size up" 派の方法論的対立は本研究で扱う重要 dimension。

## 7. 限界
- 単一症例 case report、long-term outcome 3 年のみ
- 小児 AVR 全般の RQ には外挿限定的（成人狭小弁輪と病態異なる）
- 機械弁前提なので bioprosthesis + ViV 戦略への含意は薄い
