> [!NOTE] Key Message
> 2,576例の大規模単施設コホートにて、severe PPM (iEOA ≤0.65) は **late mortality HR 1.38 (1.03-1.77)**、心血管死 HR 1.63 (1.32-2.32) を独立に増加。重要発見：**severe PPMの悪影響は <70歳・BMI <30・LVEF <50% の患者で顕著、70歳以上の高齢者・肥満者では消失**。moderate PPMはLV dysfunction患者でのみmortality増加。

---

## 1. 背景・問題設定
Pibarot labからの2,576例の大規模解析。PPMがlate survivalに与える影響を age / BMI / LV function 別に modulated されるか検証。

## 2. 患者集団
- N: 2,576（first AVR ± CABG, 1992-2005, Laval Hospital）
- 平均年齢: 68.5±10歳（severe PPM 69±11, moderate 71±9, nonsig 68±10）
- 性別: 61%男性
- BSA: 1.8±0.2 m²（severe PPM 群 1.9±0.3）
- 民族: カナダ（欧米）
- 弁種: stented bio 56%, stentless 22%, mechanical 22%; concomitant CABG 44%
- Prosthesis size ≤21mm: nonsignificant PPM 16%, moderate 38%, **severe 80%**

## 3. 手技詳細
- iEOA は projected indexed EOA（reference EOA / BSA）で決定
- Validation: 496例で術後1ヶ月Doppler echoによる measured iEOA で確認
- PPM定義: nonsignificant >0.85, moderate >0.65 and ≤0.85, severe ≤0.65 cm²/m²

## 4. アウトカム

| Outcome | Result |
|---|---|
| 30-day mortality | excluded（生存退院例のlate survival解析） |
| Stroke | NR |
| Pacemaker | NR |
| AKI | NR |
| Severe PPM | 2% (n=40) |
| Moderate PPM | 31% (n=797) |
| Mean gradient postop | nonsig 11±4, moderate 15±5, severe 21±9 mmHg |

**重要 interaction:**
- Age <70歳でsevere PPM HR 1.77 (1.10-2.58), Age ≥70 NS
- BMI <30で severe PPM HR 1.59 (1.13-2.09), BMI ≥30 NS
- LVEF <50%: moderate PPM HR 1.21 (1.01-1.41), preserved LVEFではNS

## 5. 議論ポイント
- 若年（<70歳）患者は cardiac output 需要が高く長期暴露期間も長いため、PPM感受性が高い。
- 肥満者でBSA index がEOAを過小評価（fat free mass使用提案）。
- LV dysfunction患者ではmoderate PPMでも追加afterloadが致命的。
- 管理戦略: 若年・痩せ型・LV dysfunctionでは「stentless / supra-annular / 新世代mechanical / root enlargement」を積極的に検討。

## 6. 本研究RQへの含意（最重要）
- **Primary RQ**: **若年・小体格・LV dysfunction**を持つ狭小弁輪患者は、まさにroot enlargement最重要のターゲット。本研究は「狭小弁輪 + 小BSA」の組み合わせがPPM感受性最大であることを示す（high impact）。
- **Secondary RQ（ViV戦略）**: 若年でinitial AVRが行われる場合、life-time managementでViVの可能性を高めるためにinitial label size ≥23mm（possibly 25mm）を確保する戦略を支持。Severe PPMが10年生存40%まで低下するため、initial弁サイズの確保は将来のViVだけでなく直接生存にも寄与。
- **PPM閾値の意味**: severe ≤0.65, moderate ≤0.85 がmodulator (age, BMI, LVEF)依存で意味が異なることを実証。閾値の固定的解釈ではなく**患者特性別感受性**を強調。
- **Tertiary RQ（Asian）**: BMI<30（痩せ型）でPPMが顕著影響→ アジア人（一般にBMI低い）はPPM感受性が高い可能性を強く示唆。

## 7. 限界
- 単施設・retrospective design。
- iEOA は projected （reference EOA）依存、ただし496例の measured iEOA で validation 済。
- BMI ≥30 患者数が比較的少なく、肥満subgroup の検出力に制約。
- Severe PPM n=40で多変量解析にはやや小さい。
