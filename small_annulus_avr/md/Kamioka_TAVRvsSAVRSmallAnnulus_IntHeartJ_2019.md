> [!NOTE] Key Message
> 高リスク・小弁輪（≤20mm TEE）の **日本人 94 例** で Sapien XT 23mm TAVR vs ストentedバイオプロステーゼ SAVR を比較。**TAVR で severe PPM 2.9% vs SAVR 22.0% (P=0.01)** と TAVR が有意に PPM 少。2年死亡率は同等（19.4% vs 19.2%）。**日本人小体格高齢ハイリスクで、AAE せず・SAVR で 19/21mm 弁を使うと PPM 高率**であり、TAVR は代替肢として有用。

## 1. 背景・問題設定

小弁輪 (small aortic annulus; SAA) は SAVR 後の PPM・術後死亡を増加させる古典的リスク。第一世代 THV による TAVR は SAVR より hemodynamics 良好であることが PARTNER 等で示されたが、**第二世代 THV (Sapien XT) を用いた SAA 患者の長期データは不足**。本研究は高リスク日本人 SAA における TAVR vs SAVR の hemodynamics と臨床アウトカムを比較。

## 2. 患者集団

- N=94（TAVR 35, SAVR 59）, 単施設 (小倉記念病院, 2013-2014)
- 年齢: 80.2 ± 8.6 歳（TAVR 85.1 vs SAVR 77.4, P<0.001）
- 女性: 93.6%
- BSA: 1.38 ± 0.14 m²（小体格）
- STS-PROM: 6.8 ± 4.3%
- SAA 定義: TEE で aortic annulus diameter ≤20mm（平均 19.1 mm）
- TAVR: Sapien XT **23mm を全例**（underfilled volume で展開）
- SAVR: Mosaic Ultra 19mm 32例 / 21mm 3例, Magna Ease 19mm 15例 / 21mm 9例; **57/59 supra-annular position**, 4 例は機械弁で除外
- **AAE/root replacement は SAVR 群で施行せず**

## 3. 手技詳細

- TAVR: 標準的 transfemoral or transapical、Sapien XT 23mm を SAA に対しても全例使用、annulus 破裂 0
- SAVR: 標準的胸骨正中切開, CPB. 弁輪・周囲石灰化を慎重除去。Supra-annular implantation を優先

## 4. アウトカム

| Outcome | Result |
|---|---|
| 30-day mortality | TAVR 0% / SAVR 0% |
| Stroke | TAVR 8.6% / SAVR 0% (P=0.05) |
| Pacemaker | TAVR 5.7% / SAVR 3.4% (P=0.63) |
| AKI ≥ stage 2 | TAVR 2.9% / SAVR 3.4% (P=1.00) |
| Severe PPM (iEOA<0.65) | **TAVR 2.9% vs SAVR 22.0% (P=0.01)** |
| Moderate PPM | NR（severe のみ報告） |
| Mean gradient postop | TAVR < SAVR（具体値は Figure 1）, 6 ヶ月で持続 |

| 1y mortality | TAVR 11.4% / SAVR 6.8% (P=0.43) |
| 2y mortality | **TAVR 19.4% vs SAVR 19.2% (P=0.95)** |
| 5y survival | NR |
| 10y survival | NR |

## 5. 議論ポイント

- **Sapien XT 23mm を underfilled で SAA に implant**: TAVR 群の severe PPM が他試験 (7-19.7%) より低い (2.9%)
- THV vs surgical valve の構造差（薄い stent strut, hanging deployment）が EOA を有利化
- 高リスク TAVR 群が SAVR と同等の 2y mortality → 高齢小体格における TAVR の妥当性
- SAVR 22% の severe PPM は supra-annular implantation 採用しても回避困難
- AAE/root replacement を高齢 cohort に施行せず → TAVR が代替

## 6. 本研究RQへの含意（最重要）

- **Primary RQ**: 高齢日本人 SAA で **AAE 回避 + supra-annular SAVR** では severe PPM 22% と高率。Y-incision/Manouguian 等で 1-2 サイズアップした方がよかった可能性。本研究は **「AAE しないと SAVR で何が起こるか」を定量化した日本人ベンチマーク**として極めて重要。
- **Secondary RQ (19mm 許容性, initial sizing)**: SAVR 群の弁サイズは 19mm が主体 (Mosaic 32+Magna 15 = 47/59 ≈ 80%)。BSA 1.38 m² + 19mm valve で severe PPM 22%。19mm valve は **AAE せずに使うと PPM 高率**の経験的エビデンス。
- **Tertiary RQ (Asian-specific 戦略)**: ★ **日本人 BSA 1.38 m² + annulus 19mm という小体格小弁輪の典型例**。TAVR 23mm を全例 underfilled で展開する戦略は日本独自の試み。高齢 ハイリスクでは TAVR > SAVR が現実的解。一方、low-risk/若年では AAE 併用 SAVR の意義が残る。

## 7. 限界

- 単施設、retrospective, n=94 と小規模
- 観察期間 20.6 ヶ月で短期、long-term PPM 影響は評価できず
- SAA 定義が 2D TEE であり MDCT/MRI 比 precision に劣る
- TAVR 群全例 Sapien XT 23mm（当時最小サイズ）であり、現行の 20mm THV や Evolut Pro/R との比較は不可
- AAE 適応については議論されたが施行例なし → 戦略間の直接比較不能
