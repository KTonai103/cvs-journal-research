> [!NOTE] Key Message
> VIVID registry 1,987 例の ViV-TAVR で **periprocedural PPI 発生 6.4% (128/1987)**。**新世代 THV で PPI 低下 (4.7% vs 7.4%, p=0.017)**、特に **Evolut R/Pro は CoreValve より低い (3.7% vs 9.0%, p=0.002)**。新世代 BEV と SEV の PPI rate は同等 (6.1% vs 3.9%, p=0.18)。**Independent risk factors: 高齢 (OR 1.05/yr), 大きい THV size (OR 1.10/size), 既往 RBBB (OR 2.04)**。**30日 mortality は PPI 4.7% vs no-PPI 2.7% (p=0.19, NS)、しかし follow-up で PPI 群 mortality 高い傾向 (HR 1.39, p=0.04; age 補正で p=0.08)**。ViV-TAVR の PPI は native TAVR より低く、新世代 SE で更に低減。

## 1. 背景・問題設定
- ViV-TAVR の主合併症の PPI は native TAVR と比べ data 少ない、特に新世代 THV での評価が未確立。
- 本論は VIVID registry 1,987 例で **PPI incidence・risk factor・clinical impact** を device generation 別に評価。

## 2. 患者集団
- N = 1,987（prior PPM 除外）、2007–2020。
- Mean age 77.6±9.4 yr、男性 56.2%（女性 43.8%）。
- BSA 1.86±0.24 m²、BMI 27.0±5.9。
- PPI 群（n=128）は **older (80.2 vs 77.4 yr, p<0.001)** と peripheral vascular disease 多い (28.6% vs 19.7%, p=0.017)。
- 外科弁 type：stented 83.8%。
- THV: CoreValve, Evolut R/Pro, SAPIEN/XT/3, Acurate, Portico, Lotus 等。新世代＝Evolut R/Pro, SAPIEN 3, Acurate, Portico, Lotus；旧世代＝CoreValve, Cribier-Edwards, SAPIEN, SAPIEN XT。

## 3. 手技詳細
- Periprocedural PPI 評価。
- 新世代 vs 旧世代 THV、BEV vs SEV で比較。
- Multivariable logistic regression で PPI 予測因子同定。

## 5. 議論ポイント
- ViV-TAVR の PPI rate (6.4%) は native TAVR (10–25%) より低い—外科弁 ring が conduction system を mechanical shield しているため。
- Evolut R/Pro が CoreValve より PPI 半減—frame の short skirt と repositionability の効果。
- Mortality と PPI の関係：age と PPI が両方独立予測因子だが、age 補正後 PPI の独立効果は弱まる (p=0.08)。

## 6. 本研究RQへの含意（**最重要**）
- **Primary RQ — CatF**: ViV-TAVR の PPI は native TAVR より低く、新世代 SE で更に低減（Evolut R/Pro 3.7%）。
- **Secondary RQ — initial SAVR sizing と将来 ViV**:
  - 「Larger THV size」が PPI 予測因子—**初回 SAVR で大きな弁（AAE 後）→ ViV で大きい THV → やや PPI 増加**という trade-off。ただし absolute risk 増加は小 (OR 1.10/size = ~10%/サイズ)。
  - 既往 RBBB （高齢者で多い）が独立 risk → 高齢ほど慎重に device 選択（Evolut R/Pro が望ましい）。
- **CatE 比較視点**: 初回 TAVI後の re-TAVI (ViV-in-TAVI) は本登録に含まれないが、機構的には外科弁 ring の代わりに TAVI frame 内で deploy するため、conduction shield 効果が不明—初回 TAVI 戦略は再 PPI リスクの不確実性を抱える。
- **Tertiary RQ (Asian-specific)**: Asian 高齢小体格 → 大きい THV 必要性は低い（small valve への ViV）、本論の "larger THV size → PPI" 効果は逆方向で Asian には有利。

## 7. 限界
- Observational、indication bias（device 選択は患者背景に依存）。
- Long-term PPI 影響（heart failure, mortality）の検出は power 不足。
- PPI 後の pacing % data なし、機能的影響不明。
