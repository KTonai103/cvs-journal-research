> [!NOTE] Key Message
> SURD-IR 国際登録の **BAV サブグループ 191 例** (Perceval 46.1% / Intuity 53.9%)。**30日死亡 1.6%**、stroke 4.2%、PMI 7.9%、moderate-severe AR 3.7%。Type 0 BAV (raphe なし) は禁忌、type 1 が主適応。**commissural plication + 完全石灰除去**で sealing 確保可能。小弁輪 (19/21 mm) でも Intuity 比較的良好な hemodynamics (mean gradient 19 mm: 14.1 ± 5.2 mmHg)。**BAV は SURD 絶対禁忌ではないが、解剖把握と technical caveats が必須**。

---

## 1. 背景

BAV は AS-AVR の 50% 以上を占めるが、root geometry の非対称・elliptic annulus・commissural height 不均一が SURD 弁の sealing/migration リスクを上げる。これまで BAV は SURD の relative contraindication とされてきた。本研究は SURD-IR の BAV サブグループで安全性・血行動態を評価した。

## 2. 患者集団

- SURD-IR 全体 4,636例から BAV のみ 191例 (4.1%) を抽出。
- 男性 48.2%、平均年齢 70.7 ± 8.1 y、BMI 26.6 ± 4.5。
- EuroSCORE I 5.8% (3.3–8.7), II 1.6% (1–2.6)。
- 民族: ヨーロッパ・豪州・カナダの 18 施設 (Asian なし)。
- Sievers 分類は database 内で扱えず、type 0/1/2 別の AR rate は分析不能。

## 3. 手技

- アプローチ: full sternotomy 26.2%、ministernotomy 47.6%、right anterior mini-thoracotomy 26.2% (**73.8% が minimally invasive**)。
- 弁: Perceval 88例 (46.1%)、Intuity/Intuity Elite 103例 (53.9%)。
- 併施手術 31.9% (CABG 13.6%, 胸部大動脈 13.5% 等)。
- Technical key points: (i) 360° decalcification, (ii) commissural plication で annulus を三尖様に circular 化, (iii) CT で root geometry 評価, (iv) sizing は under/over とも sealing/PMI に影響。Type 0 は **絶対禁忌**。

| Outcome | Result |

| 30-day mortality | 1.6% (3/191) |
| Stroke (isolated AVR) | 3.8% |

## 5. 議論ポイント

- BAV の幾何学的非対称が SURD 弁 sealing の最大障壁。Type I Sievers (1 raphe) は3尖化 remodeling 可能、Type 0 は不可。
- Stroke 率 4.2% は他 SURD 報告より高い → BAV calcification の debridement が原因と推定。
- PMI 7.9% は標準 AVR より高いが SURD 全体平均と同等。Decalcification + LVOT への弁拡張圧が機序。
- **小弁輪との関連**: 19mm Intuity の mean gradient 14.1 mmHg は同サイズ stented 弁より明らかに低い → **狭小弁輪+BAV** の症例で AAE を要さず SURD で対応する選択肢を示唆。
- Sizing の慎重さ強調: 小さすぎると AR、大きすぎると PMI / hemodynamic turbulence。

## 6. 本研究RQへの含意

- **Primary RQ (AAE vs no-AAE vs TAVI)**: BAV + 狭小弁輪症例で、**SURD は AAE 不要で大 EOA を確保できる第3戦略** として位置付けられる。ただし type 0 BAV は除外。
- **Secondary RQ (PPM/AR/PMI)**: PMI 7.9% は通常 AVR より高 → 高齢者で許容、若年で要慎重。AR 3.7% は acceptable。
- **Tertiary RQ (Asian/小体格)**: 欧州 BMI 26.6 集団、日本人 BAV+小体格には外挿に注意。Intuity 19mm の hemodynamic data は最も小弁輪に近い参照値として有用。
- **AAE 代替議論**: 小弁輪 BAV で commissural plication + SURD は AAE 不要の合理的選択。ただし type 0 BAV は SAVR + AAE が安全。

## 7. 限界

- 後方視的、core lab なし、follow-up なし → 遅発性 migration/SVD 不明。
- Sievers 分類 unrecorded で AR と type の関連を解析不能。
- Paravalvular vs central regurgitation 区別不能。
- PPM rate (iEOA) 直接報告なし、gradient のみ。
