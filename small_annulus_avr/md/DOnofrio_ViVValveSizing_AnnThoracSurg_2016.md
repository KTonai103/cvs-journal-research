> [!NOTE] Key Message
> Italian multicenter ViV cohort 65 例（VIV-A 44, VIV-M 22）。**30日死亡 6%（VIV-A 4.5%, VIV-M 9%）、1/2/3/4年生存 84.4%/80.5%/74.3%/62%**。**外科弁 internal diameter (ID) <20mm では postoperative gradient が ID 21–23mm および >23mm より有意に高く、severe stenosis (mean gradient >35 mmHg) 3例 (6.8%) はすべて ID <20mm 群**。**「ViV では初回 SAVR の ID ≥21mm（label ≥23mm 相当）が hemodynamic 成功の閾値」** という具体的 sizing recommendation を提示—本研究 RQ の「initial SAVR で何 mm 以上を植えるべきか」に直接答える論文。

## 1. 背景・問題設定
- ViV-TAVR が再開胸 SAVR の代替として普及するも、外科弁サイズと post-ViV hemodynamics の関係は十分定量化されていない。
- 本論は VIV-A と VIV-M（mitral）の clinical/hemodynamic outcome を multi-center で評価し、特に **stent ID と postop gradient の関係** を主検討。

## 2. 患者集団
- N = 65（VIV-A 44, VIV-M 22, 1 combined）。
- VIV-A: 平均年齢 77±10 yr、男性 57%、BSA 1.74±0.21 m²。
- LogEuroSCORE 28±16%。
- 失敗 mechanism (VIV-A): stenosis 36%, regurgitation 20%, mixed 43%。
- **本研究 RQ にとって最重要：外科弁 ID 層別**:
  - ID <20mm
  - ID 21–23mm
  - ID >23mm
  - （内訳具体数値は table 内で示される、small ID 群の n は本文断片では明示せず）

## 3. 手技詳細
- Access: TF 28 (43%), TA 37 (57%)。VIV-A: TF 62%。VIV-M: 100% TA。
- Device: CoreValve 12 (VIV-A のみ), SAPIEN/XT 54。
- SAPIEN サイズ（VIV-A）: 23mm 22 (50%), 26mm 8 (18%), 29mm 2 (4%)。
- CoreValve サイズ: 23mm 6, 26mm 4, 29mm 1, 31mm 1。
- Sizing: stent ID (manufacturer chart) と true ID (Bapat chart) 併用、stentless は CT で annulus 直接測定。

## 5. 議論ポイント
- ID <20mm では severe stenosis 6.8% で発生、全例この群—**ID 20mm が ViV hemodynamic 成功の critical cutoff**。
- 21mm label の Carpentier-Edwards Perimount などは true ID ~19mm のため "small valve" に該当。
- BE と SE 混在で device-specific 解析不能だが、ID <20mm では device 問わず gradient 高い。
- Diabetes が strong predictor (HR 7.2) — patient comorbidity も重要。

## 6. 本研究RQへの含意（**最重要**）
- **Primary RQ**: ViV-TAVR feasible だが **stent ID <20mm（label ≤21mm 相当）では severe stenosis のリスクが顕在化**。
- **Secondary RQ — initial SAVR の valve sizing で将来 ViV を許容するための閾値**:
  - **初回 SAVR で stent ID ≥21mm（label size ≥23mm が多く対応）を植え込むべき**。
  - 21mm label SAVR (true ID ~19mm) は ViV で severe gradient リスク。
  - **AAE/Y-incision で 23mm 以上を確保する戦略の根拠**。
- **本論は本研究 RQ への直接寄与が最大の論文の 1 つ**：「19mm/21mm 弁植込みの future ViV 許容性」に対し定量的回答。Asian 小体格で AAE なしに 19mm 植込んだ場合、本論の Italian cohort より gradient 不利の可能性。
- **Tertiary RQ (Asian-specific)**: Italy cohort、Asian 該当なし。Asian で同様の study が必要、ただし本論の閾値（ID 20mm）は body-size を超えた生物物理的限界に近い。

## 7. 限界
- N=44 (VIV-A) と限定的、small ID subgroup の event 数も少ない。
- BE/SE 混在、旧世代 device 中心。
- Follow-up 14±14ヶ月、long-term durability 未評価。
- Multicenter retrospective、selection bias 残存。

## 8. 引用文献ハイライト
1. **Bapat 2014 (J Heart Valve Dis)** — true ID chart、本論 sizing の基盤。
2. **Dvir 2014 (JAMA, VIVID)** — small ≤21mm 弁で 1y mortality 25%、本論と整合。
3. **Simonato 2016 (Circ Cardiovasc Interv)** — VIVID、high implantation で gradient 低減（CoreValve 0–5mm, Sapien 0–10% 推奨）。
