> [!NOTE] Key Message
> JCS/JSCS/JATS/JSVS 2020 VHD Guideline (日本初の総合 VHD guideline)。SAVR vs TAVI 選択は **age cutoff を <75 SAVR / ≥80 TAVI** で提示するも、"明確な cutoff を設けない" と明言 (日本の長寿命を考慮)。**Table 33 (SAVR/TAVI 選択因子) で "Small annulus highly expected patient-prosthesis mismatch" を TAVI を支持する factor として明記** — これは JCS 独特の表現で、**狭小弁輪+PPM 予測 → TAVI** という方向性を ACC/AHA Decision Table と逆の論理で示す。**PPM 定義は JCS 独自に BMI <30: iEOA <0.65 cm²/m², BMI >30: <0.55 cm²/m²** に細分。**ViV を見越した初回 SAVR の弁選択の重要性** を narrative で強調 (line 8372-8374)。**Valve-in-valve の poor prognosis in small bioprostheses with PPM** を明確に警告 (line 9138-9139)。**AAE への直接的 Class/LOE 推奨はない**が、初回 SAVR で大きい弁を入れる戦略は narrative 推奨。

## 1. 背景

日本の VHD 診療 guideline の最新版 (2017 改訂版を統合・拡張)。日本特有の課題:
- TAVI 導入 (2013年)、ViV 適応 (2018年) を反映
- 日本人の **長寿命** (≥75y で米欧より +1-2y)
- 日本人の **小体格** (BSA <1.6-1.8 が多い)
- バイオ弁使用率増加 (Japan Cardiovascular Surgery Database 2013-14: 50代 20%, 60代 60%, ≥70歳 >90%)

## 2. 対象範囲

成人 VHD 全般。AS / AVR / 弁種選択 / PPM / ViV / 高齢者の特別考慮を含む。本MDでは AS / AVR / 狭小弁輪・PPM 関連箇所のみ抽出。

## 3. ガイドライン構成

- I. 共通事項 (Heart Team, 検査, リスク評価, 高齢者特別考慮)
- VI. AS (5.3 Intervention の indications)
- XII. Prosthetic Valve Replacement (1.3 Valve selection Table 46, 2.4 ViV)
- CQ3: Asymptomatic very severe AS 早期手術 ← Clinical Question 章

## 4. 主要推奨

### 4.1 AS Intervention Indications (Table 32, line 5832)

| 推奨 | Class | LOE |
|---|---|---|
| Symptomatic severe AS → intervention (SAVR or TAVI) | I | B |
| Asymp severe AS + LVEF <50% → intervention | I | C |
| Asymp severe AS + other cardiac surgery → SAVR | I | C |
| Asymp severe AS + symptoms on exercise → intervention | I | C |
| Asymp severe AS + BP↓ on exercise → intervention | IIa | C |
| **Asymp very severe AS (V ≥5, mPG ≥60, AVA ≤0.6) + low risk** | **IIa** | **B** |
| Asymp severe AS + severe PHTN (PASP ≥60) + low risk | IIa | C |
| Asymp severe AS + rapid progression (ΔV ≥0.3/y) + low risk | IIb | C |
| Asymp moderate AS + other cardiac surgery | IIa | C |

### 4.2 SAVR vs TAVI の指針 (Section 5.3.1, line 5979-5983)

> "**Given the various factors to consider and the lack of definitive evidence, the present guidelines do not establish a clear cutoff value of age for TAVI or SAVR. Rather, the present guidelines offer an index of prioritization that suggests using TAVI in patients aged ≥80 years and SAVR in those aged <75 years.**"

**日本特異性 (line 5962-5966)**:
> "In Japan, the average life expectancy is known to be 3–5 years longer than that in Europe and the USA, and life expectancy of the elderly in Japan aged ≥75 years is 1–2 years longer than that in the USA and Europe."

→ ESC/EACTS の 75y cutoff より TAVI 推奨年齢を引き上げる方向 (≥80 y で TAVI 優先) — 日本の長寿命を考慮した独自姿勢。

| Factor (要約) | SAVR | TAVI |

| Suspicion of endocarditis | ✓ | |
| Concomitant CABG / mitral / TR / aortic aneurysm / septal myectomy 必要 | ✓ | |

| Liver cirrhosis / COPD (FEV1.0 <1L) / Interstitial pneumonia / bleeding tendency | | ✓ |

**★ Line 6078-6079**: JCS は **"Small annulus highly expected PPM" を TAVI を支持する factor** として位置付ける。これは Vahanian 2021/2022 (Table 6 で AVA <0.65 cm²/m² → SAVR favor) と **明確に逆の論理**。背景は日本人小体格で SAVR で弁を大きくしても PPM が出やすい現実 + AAE が日本で限られた施設でしか行われない現実に基づくと考えられる。

### 4.4 Prosthetic Valve Selection (Table 46, line 8250)

| 推奨 | Class | LOE |
|---|---|---|
| Informed consent で最終決定 | I | C |
| 抗凝固不可・コンプライアンス不良 → bioprosthesis | I | C |
| 妊娠予定の女性 → bioprosthesis | IIa | C |
| **AVR <60 y → mechanical valve** | **IIa** | **C** |
| **AVR ≥65 y → bioprosthesis** | **IIa** | **C** |
| Re-do for mechanical valve thrombosis → bioprosthesis | IIa | C |

- 60–65y aortic は両方 acceptable。
- 透析患者: bioprosthesis durability 問題既知だが最近研究では差なし。**JCS は推奨を出さず case-by-case**。
- Japan Cardiovascular Surgery Database 2013-14 (line 8312-8315): 50代 20% bio、60代 60%、≥70 >90% bio。
- **Line 8372-8374 (narrative 推奨)**: "**Transcatheter intervention for SVD after SAVR (valve-in-valve) is now common, and the age cutoff for bioprostheses may become lower in the future. In addition, the selection of valve (size and type) at first SAVR is important, given the potential for future valve-in-valve procedures.**" → **初回 SAVR で大きい弁を入れる戦略 (AAE 含む) の重要性** を narrative で強調。

### 4.5 PPM Definition (JCS 独自の BMI 調整、line 9085-9088)

> "Severe PPM is defined as an EOA index <0.65 cm²/m² (for BMI <30 kg/m²) or <0.55 cm²/m² (BMI >30 kg/m²) at the aortic position, or an EOA index <0.90 cm²/m² (BMI <30 kg/m²) or <0.75 cm²/m² (BMI >30 kg/m²) at the mitral position."

→ 国際 Pibarot cutoff の BMI 補正版を採用。

### 4.6 ViV と狭小弁輪 (Section 2.4.3, line 9138-9143)

> "**There are also reports of poor prognoses when valve-in-valve is performed for small-sized bioprosthetic valve cases of PPM just after initial SAVR. Re-do SAVR or valve-in-valve options should be determined on a case-by-case basis by the Heart Valve Team with consideration of the size and type of bioprosthesis, anatomical compatibility of the size of the Valsalval sinus, surgical risk, etc.**"

→ **初回 SAVR で小さい bioprosthesis + PPM だった症例の ViV は予後不良** と明示 → 初回 SAVR での弁拡大戦略 (AAE / 大サイズ弁) の重要性を補強。

### 4.7 ViV TAVI 推奨 (Section 2.4.3)

| 推奨 | Class |
|---|---|
| Severe bioprosthetic stenosis/regurgitation + HF → re-do SAVR | I |
| Severe regurgitation + severe hemolysis → re-do | I |
| **Symptomatic severe aortic bioprosthetic stenosis/regurgitation + high surgical risk → ViV TAVI** | **IIa** |

### 4.8 高齢者特別考慮 (Section I.6)

- 日本人の平均余命延長 (line 1477-1489)
- 高齢者は age だけで判断せず **frailty + comorbidity** を統合的に評価
- TAVI vs SAVR は Heart Valve Team で十分議論
- 寝たきり・認知症患者への invasive 介入は予後不良 → 厳格に回避

### 4.9 Sutureless valve への言及 (Section XII)

- Line 8773 で "stentless Solo Smart and sutureless valves" を SVD の文脈で短く言及 — 詳細推奨なし。
- AAE への明文 Class/LOE 付与は確認できず — narrative 内に root enlargement は登場せず (Beckmann 等の data に依拠していない様子)。

## 5. 議論ポイント

- **JCS 最大の特徴 1**: Table 33 で **"Small annulus + PPM 予測 → TAVI"** と明示 → これは **ACC/AHA Exec Summary** ("annular enlargement might be considered" → SAVR を favor) と **真っ向から対立**。
  - 解釈: 日本では AAE 経験を持つ施設・術者が限定的、日本人小体格で SAVR の PPM 回避が技術的に困難 → TAVI への流れを正当化。
  - しかし、これは **本研究 RQ の核心**: 日本でも AAE の経験を積めば狭小弁輪+若年症例で SAVR + AAE が TAVI より有利となる可能性を示唆 (Yang 2024 JTCVS Y-incision 等の new technique の余地)。
- **JCS 最大の特徴 2**: 初回 SAVR の弁サイズ・タイプ選択を ViV 適格性のために重視 (line 8372-8374) → AAE の長期戦略的意義を間接支持。
- **JCS 最大の特徴 3**: 日本人長寿命を反映し TAVI 推奨年齢を **≥80 y** に引き上げ — ESC/EACTS の 75y より保守的に SAVR 領域を確保。
- **PPM 定義**: BMI 別の cutoff は国際 guideline で珍しい。日本人小体格 (BMI <25 が多い) では <0.65 cm²/m² 基準 → やや厳しめの判定となる。
- **SURD への姿勢**: 言及最小限。Vahanian 2021 ("RCT 不足") と類似の保守的姿勢。

## 6. 本研究RQへの含意

- **Primary RQ (AAE-SAVR vs no-AAE vs TAVI)**:
  - **JCS は狭小弁輪 + PPM 予測 → TAVI を推奨する唯一の major guideline** (Table 33)。これは日本の臨床現実を反映するが、本研究の motivation そのもの: **日本人小体格・狭小弁輪でも若年症例は AAE-SAVR が長期予後上 TAVI より良いはず**。
  - <75 y で SAVR 推奨、<60 y で mechanical 推奨 → **若年小体格症例で AAE-SAVR + mechanical** が JCS guideline 内でも合理的選択肢として残る。
  - 一方、≥80 y で TAVI 優先 + 狭小弁輪 → TAVI 強推奨。65–80 y の equipoise 領域が真の議論点。
- **Secondary RQ (PPM/regression/予後)**:
  - PPM 定義 (iEOA <0.65/<0.55 BMI 別) は本研究の operational definition と一致 (small_annulus_avr/CLAUDE.md の cutoff)。
  - ViV poor prognosis after PPM の警告 → **初回で大きい弁を入れる (AAE) ことの長期価値** を直接補強。
- **Tertiary RQ (Asian/小体格)**:
  - **JCS は本研究の最重要 reference**: 日本人小体格・長寿命・bio 弁使用増加の現状を guideline 化。
  - BSA <1.6-1.8 の日本人女性の AS-AVR 推奨枠組み確立。
- **AAE 代替議論 (sutureless)**: JCS は SURD に明示推奨なし。日本での SURD 採用が限定的な現状を反映。

## 7. 限界

- AAE への Class/LOE 付与なし、narrative も限定的 → 日本での AAE 採用の institutional 推奨が弱い。
- SURD valves への詳細推奨なし。
- Table 33 が "Small annulus + PPM → TAVI" を推奨する一方、初回 SAVR の弁拡大も narrative で支持 → 内部に若干の論理的乖離 (狭小弁輪症例で TAVI を選ぶか SAVR + 大弁を選ぶかが不明確)。
- Yang 2024 (Y-incision) 等の新 AAE 技術への言及なし (2020 公開のため)。
