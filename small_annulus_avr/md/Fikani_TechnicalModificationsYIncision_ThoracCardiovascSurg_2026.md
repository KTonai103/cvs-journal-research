> [!NOTE] Key Message
> Beirut (レバノン) チームからの Y-AAE 技術修正 3 点: (1) Dacron → **bovine pericardial patch** (more pliable, suture しやすい) (2) 最初 left trigone anchor 後に **patch を心室内へ inversion** し suture line を露出 (3) 外側 **bovine pericardial reinforcing strip** を aortic root に走行させ hemostatic security 強化。Univ Michigan 以外初の他施設 modification 報告。

---

## 1. 背景・問題設定
Y-incision AAE は世界に広まりつつあるが、Dacron patch の rigidity と suture line の hemostasis (特に unclamp 後) が課題。Beirut の Jebara チームが institutional refinement を提案。

## 2. 患者集団
- Institutional series of "their practice"、具体的 n・demographic 未開示

## 3. 手技詳細
3 つの modification:
1. **Patch material**: Dacron → **bovine pericardial patch** (pliable で suture 易)
2. **Suturing strategy**: 左 trigone での最初 anchoring stitch を結紮 → patch を ventricle 内に **invagination** → suture line と annulus alignment 視野良好 → 右 trigone へ running 完了
3. **External reinforcement**: aortic root 外側に **bovine pericardial reinforcing strip** を走行縫合、unclamp 後の hemostasis 向上、追加 hemostatic maneuver 必要性低下

## 5. 議論ポイント
- Bovine pericardium は Hemashield Dacron より pliable で curvilinear suture が容易
- Patch ventricle 内 invagination は suture exposure の technical innovation
- External reinforcing strip は aortic root の通常 hemostasis-prone area で予防的補強
- 基本 framework (Y-incision + rectangular patch) は維持

## 6. 本研究RQへの含意（最重要）
- **Primary RQ**: 狭小弁輪 AVR の Y-AAE 適用において **patch material と hemostasis strategy の改良**を提示。Univ Michigan モデルの普遍性と adaptation 可能性を示唆。
- **Secondary RQ (ViV用 initial sizing)**: 直接記載なし、ただし simplification によって surgeon adoption が広がれば routine large-prosthesis 留置の dissemination に寄与。
