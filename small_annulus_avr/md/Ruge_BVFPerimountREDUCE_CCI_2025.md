> [!NOTE] Key Message
> **REDUCE registry：6 欧州施設、Perimount P 2900 への ViV-TAVI 240 例**。**True ID ≤21mm が 60% (144/240)**、SEV 60%、BEV 40%。**30 日生存 93.3%**、VARC-3 device success 71%。**Multivariable 回帰**で post-ViV mean gradient の独立予測因子は **(1) surgical valve size: 1mm 増 → −0.84 mmHg (p=0.001)、(2) postdilation 施行: −3.25 mmHg (p=0.007)、(3) SEV 使用: −7.31 mmHg (p<0.001)**。**驚いたことに BVF は "standard" postdilation と比べ hemodynamics 優位を示さず**、結論として **「Perimount small ID ≤21mm への ViV では SEV + postdilation の組合せが gradient 最適化、BVF の付加効果は限定的」**。

---

## 1. 背景・問題設定
- ViV-TAVI for failed Perimount は普及するが、small surgical valve (label ≤21mm, true ID ≤20mm) で gradient 高、1年 mortality も悪い (Dvir 2014)。
- BVF は in vitro で gradient 低減効果が確認、in vivo の比較 data は限定的。
- 本 REDUCE registry は failure mode、device 選択、postdilation 戦略（no / standard / BVF）の比較で gradient を解析。

## 2. 患者集団
- N = 240、6 欧州施設、ViV-TAVI in failing Perimount P 2900。
- Median age 78 (IQR 70–83)、男性 66%、logistic EuroSCORE 20%。
- BMI median 26、creatinine 1.1 mg/dL。
- **True ID ≤21mm: 144/240 (60%)**。
- Failure mode: stenosis 42%, regurgitation 10%, mixed 48%。

## 3. 手技詳細
- THV: SEV 60% (Evolut R/Pro/Pro+, ACURATE), BEV 40% (SAPIEN 3/Ultra)。
- 3 群:
  - No postdilation: 116 (48%)
  - Standard postdilation: 88 (37%)
  - BVF: 36 (15%)
- BVF: high-pressure non-compliant balloon (True Dilatation/Atlas Gold), ID +1mm 以上、rapid pacing。
- VARC-3 endpoints。

## 5. 議論ポイント
- **SEV の gradient 優位 (−7.31 mmHg)** が device 因子としては最大。
- **Postdilation の有無 (−3.25 mmHg)** が次。**BVF と standard postdilation 間で hemodynamics 差なし**（surprise finding）。
- 外科弁 size 1mm 増 → 0.84 mmHg 改善：small valve のペナルティ定量化。
- Mean time SAVR → ViV-TAVI は 11.7±4.0 年、典型的な Perimount durability。

## 6. 本研究RQへの含意（**最重要**）
- **Primary RQ — CatF 中心**: ViV-TAVI for Perimount は **「SEV + postdilation」が gradient 最適化の鍵**、BVF は追加 benefit なし。
- **Secondary RQ — initial SAVR sizing と将来 ViV**:
  - **Surgical valve size 1mm 大 → −0.84 mmHg** という線形効果が確認 → 初回 SAVR で AAE して 19mm → 23mm（4mm 大）植えると ViV で −3.4 mmHg 改善見込み。
  - **「True ID ≤21mm が 60%」も Perimount の現実**—Perimount 21mm label = true ID ~19mm で当てはまる。
  - 結論：**初回 SAVR で AAE して 23mm以上の Perimount-class valve を植えることが ViV で gradient 改善に直結**。
  - BVF への過度な期待は禁物—**初回 sizing で勝負する戦略が現実的**。
- **CatE 比較視点**: 初回 TAVI vs SAVR で考えれば、初回 TAVI 後の再 ViV (TAVI in TAVI) では「surgical valve」の代わりに「TAVI frame」内で postdilate 必要、本 registry 知見の外。
- **Tertiary RQ (Asian-specific)**: Asian 小弁輪 + Perimount 19mm は典型的—本研究の small ID 60% コホートと一致。Asian で initial AAE 戦略の意義が再強調。

## 7. 限界
- N=240 で BVF 群 36 (15%) は power 不足、BVF "no effect" は有意差検出不能の可能性。
- 30日 follow-up のみ、long-term hemodynamics・mortality 未評価。
- Perimount のみ、他弁 (Mosaic, Trifecta, Magna) で結果異なる可能性。
- BVF operator dependence、施設間 variability。

## 8. 引用文献ハイライト
1. **Dvir 2014 (JAMA VIVID)** — small valve ≤21mm で 1y mortality 高。
2. **D'Onofrio 2016 (Ann Thorac Surg)** — ID <20mm で gradient 高、本論の small valve 60% と整合。
3. **Simonato 2016 (Circ Cardiovasc Interv)** — high implantation + SEV で gradient 低減、本論の SEV 効果と一致。
4. **BVF 既報 (Allen 2017, Brinkmann 2021 等)** — BVF 個別 study との比較必要。
